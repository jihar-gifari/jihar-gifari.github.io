Data Science Projects

Here are several data science projects that I have undertaken before. Please note that some of these projects may contain confidential information; therefore, there might be some projects that are not explained in great detail.

Divorce Probability Prediction

- Personal Project

As a respond to the increasing divorce rates in Indonesia, this project set out to create an interpretable machine learning model. This model, boasting an impressive average cross-validation accuracy of 97%, not only predicts divorce probability but also provides users with explanations for its forecasts. Now available online, it empowers individuals and couples to make informed decisions regarding their relationships.

User Churn Probability Prediction

- Shopee Related

To address finance and marketing team concerns, this project focused on predicting user churn. The goal was to create a precise machine learning model for personalized marketing, resulting in an 82% accurate model. Shopee can now proactively target at-risk users, reducing churn rates, and boosting customer retention for cost savings and revenue growth.

Socio-Economic Status (SES) Classification

- Shopee Related

Initiated to discern potential users' SES (Socio-Economic Status) levels and refine monetization approaches for high SES users with lower spending habits, this project set out to construct a machine learning model for classifying Shopee buyers into SES groups. The outcome yielded an accurate ML model with an 80% classification accuracy, now deployed globally. This model plays a pivotal role in tailoring marketing and monetization tactics for distinct SES segments, consequently boosting revenue and enhancing customer satisfaction.

Retained Buyer Prediction

- Shopee Related

In ecommerce industry, we run a lot of campaigns everyday. But, which campaigns resulting high-quality buyer? is it only temporarily attract new buyer which will churn after the campaigns over? In this project, I created a Machine Learning Model to predict whether a new buyer will later become a retained buyer or not. So that we could assess a certain campaign immediately.

Customer Segmentation & Order Cancellation Prediction

- Personal Project

Using ecommerce data that contain an actual transaction from online retail in UK, this projects is conducted to perform a clustering and classification model. The aim of this project is to find a pattern from customer buying behavior and generates order cancellation predictions.

Market Basket Analysis

- Personal Project

This projects is one of the learning materials when me and other data professionals from Xeratic gave a corporate training and data education to one of their client, Direktorat Jenderal Pajak Indonesia. This is an unsupervised learning projects which aims to uncover patterns about which products that should be sold together. Using an actual online retail data from UK, This projects is done by using the mlxtend library and using the apriori algorithm also the association rules concepts.
